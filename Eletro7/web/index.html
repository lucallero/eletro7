Cifrador RSA para BPag

O cifrador responde ao método POST na seguinte URL: http://catalogobrasilia.com.br/eletro7/bpag
Os seguintes paramentros devem ser informados pelo POST:
cardN=número do cartão, com 16 digitos numéricos;
cardV=código de verificação, com 3 digitos numéricos;
A servlet retorna um texto em ASCII contendo subsequentemente o número de cartão e o código verificador, separados pelo caractere '#'.
Os valores são criptografados conforme exemplo fornecido para o desenvolvimento.
